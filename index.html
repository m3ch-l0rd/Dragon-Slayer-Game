About Dragon Slayer

The core concept of this project is from the freecodecamp java script project "building a role playing game...". I thought this project was very cool , it inspired me to further develop the concept, and make a webpage for it.

I used deepai image generator to generate the pixel background and character images. Some story details were edited to make it more my own and create more context for the player. I plan to continue the development of this project and my coding skills.
